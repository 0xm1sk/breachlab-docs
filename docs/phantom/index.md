# Phantom Track: Post-Exploitation & Tradecraft

Phantom starts where Ghost ends. Once you have a shell, the real operation begins. This track focuses on post-exploitation in a modern 2026 environment.

## The Five Acts of an Operation

**I. Escalation (0–9)**
Mastering the privesc arsenal: SUID/GTFOBins, sudo misconfigs, library hijacking, capabilities, and kernel exploits.

**II. Harvest & Persist (10–15)**
Turning a foothold into a permanent presence. Credential harvesting, persistence (PAM, systemd), and anti-forensics.

**III. Lateral Movement (16–19)**
Moving through the network. SSH tunneling, ligolo-ng, internal recon, and multi-host pivot chains.

**IV. Container & Cloud (20–26)**
Breaking the sandbox. Docker escapes, Kubernetes cluster takeover via service accounts, and cloud IAM pivots.

**V. Operations (27–31)**
The final stage. Custom tooling, data exfiltration (DNS/HTTPS/ICMP), and multi-host cleanup.
